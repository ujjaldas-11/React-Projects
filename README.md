## Overview

RecipeHub is a clean and intuitive recipe exploration app that helps you discover the perfect dish for any occasion. Browse categories based on your mood, check out popular featured recipes with details like difficulty, prep time, and ratings, and stay inspired with weekly updates.

Perfect for beginners and seasoned cooks alike!
